# Project Context & Roadmap

The following is a list of features and ideas to refer back to during future sessions:

- Add little micro animations for touches of the ball, or where the ball strikes a collision area
- Add little circular feet for each player so it looks like they are running/kicking
- Add unlocks for legend players/player selection for a match
- Add game timer (3min/5min)
- Add player data to make teams have different stats (speed, shooting, power, etc)
- Add some sort of gyro function for shooting or skill move or passing or something to make the game more interactive
- Refine SVG goal animation, should feel hand drawn
- Refine UI and use crude hand drawn logos or flags for each team
- Add story mode which lets you play out based on group stages. The final should get an updated pitch to put it in a stadium
- Rather than a green field, should add tiny little ui grass elements to suggest the field is grass
